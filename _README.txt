This program builds a red-black tree from the numbers.txt file and processes it in the way outlined in the __Prog2 document. 

To compile: please type the following in a linux terminal: make
To run: please type the following in a linux terminal: ./a.out

Rules for the numbers.txt file
    1.) seperate numbers by A SINGLE COMMA ONLY
    2.) no decimal values - Any decimal value will be truncated

Example numbers.txt file:
45,16,8,2,3,7,33,12
